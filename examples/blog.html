About this example

This example showcases parallax scrolling within text using relative positioning. It also demonstrates a drawback of parallax scrolling which is that if the element cannot be scrolled to its home position all the parallax child elements will never be able to get completely into position. Hense the huge padding on the bottom of this page.

All the rest of the text is standard lorum ipsum.
